## 系统概述

### 项目背景
本系统开发于能源行业数字化转型时期,针对江苏省独立储能项目的融资分析痛点:
- **问题1**: 储能项目收益计算复杂,需专业财务人员
- **问题2**: 不同季节、不同运营策略差异大,难以快速评估
- **问题3**: 融资机构缺乏标准化评估工具

### 解决方案
提供一个**即用型的在线计算工具**:
- 无需安装,浏览器即可使用
- 支持多种运营场景和政策参数
- 快速生成财务评估报告
- 全球40+语言支持

### 核心价值
| 角色 | 价值 |
|-----|------|
| 项目开发商 | 快速评估项目可行性,支撑融资决策 |
| 融资机构 | 标准化评估工具,降低风险评估成本 |
| 投资机构 | 快速对标行业收益水平 |
| 政府部门 | 产业数据分析和政策效果评估 |

## 功能模块

### 1. 参数输入模块

#### 输入字段定义

```typescript
interface StorageProjectParams {
  // 基本参数
  installedCapacity: number;    // MW - 装机容量
  storageHours: number;         // MWh - 储能时长
  chargeDischargeEfficiency: number;  // % - 充放电效率
  
  // 市场参数
  peakPrice: number;            // ¥/MWh - 峰值电价
  valleyPrice: number;          // ¥/MWh - 谷值电价
  basePrice: number;            // ¥/MWh - 基础电价
  
  // 运营参数
  operatingCyclesPerYear: number;  // 次/年 - 年运作周期数
  summerMonths: number;         // 月 - 夏季运营月数 (通常6月-9月)
  
  // 政策参数
  subsidyPrice: number;         // ¥/MWh - 补贴价格
  subsatyYears: number;         // 年 - 补贴年限
  
  // 成本参数
  annualOMCost: number;         // ¥/MW/年 - 年运维成本
  taxRate: number;              // % - 税率
  depreciationRate: number;     // % - 折旧率
}
```

#### 输入验证规则

```
装机容量: 0.1 ~ 1000 MW

效率: 50% ~ 99%
电价: 0 ~ 1000 ¥/MWh
运营周期: 0 ~ 365 次/年
其他成本: 非负数
```

## 计算模型

### 基础计算公式

#### 1. 夏季调峰收益

```
夏季收益 = (
  装机容量 (MW) × 
  (峰值电价 - 谷值电价) (¥/MWh) ×
  储能时长 (MWh) ×
  充放电效率 (%) ×
  夏季运营周期数
)

其中:
  夏季运营周期数 = 全年运营周期 × (3~4个月 / 12个月)
```

#### 2. 非夏季基础调度收益

```
非夏季收益 = (
  装机容量 (MW) ×
  基础价格差 (¥/MWh) ×
  储能时长 (MWh) ×
  充放电效率 (%) ×
  非夏季运营周期数
)

其中:
  基础价格差 一般低于峰谷价差
  非夏季运营周期数 = 全年周期 - 夏季周期
```

#### 3. 政策补贴收益

```
政策补贴 = (
  储能能量容量 (MWh) ×
  政策补贴单价 (¥/MWh) ×
  补贴年限
)

说明:
  - 江苏新增储能补贴通常为 0.3~0.5 ¥/kWh
  - 补贴通常持续3~5年
```

#### 4. 年度成本

```
年度运维成本 = 装机容量 (MW) × 年单位费率 (¥/MW)

年度税收 = (
  (夏季收益 + 非夏季收益 + 政策补贴) ×
  税率 (%)
)

成本合计 = 运维成本 + 税收 + 其他成本
```

#### 5. 年度净收益

```
年度净收益 = (
  夏季收益 +
  非夏季收益 +
  政策补贴 -
  运维成本 -
  税收
)
```

### 敏感性分析

#### 参数变化影响分析

```
对于每个关键参数,计算±偏差下的收益变化:

收益敏感性 = (
  (新参数下收益 - 基准收益) / 基准收益
) / (参数变化比 / 基准参数值)

范例:
- 电价敏感性: 电价±10% → 收益变化 ±10%
- 效率敏感性: 效率±5% → 收益变化 ±5%
- 周期敏感性: 周期±20% → 收益变化 ±20%
```

### 投资决策指标

#### 静态指标

```
单位容量收益 = 年收益 / 装机容量 (MW)

单位能量收益 = 年收益 / 能量容量 (MWh)

收益率 = 年收益 / 投资总额 (%)
```

#### 动态指标

```
内部收益率 (IRR):
  0 = -投资额 + Σ(第n年净收益) / (1+IRR)^n
  
净现值 (NPV) @折现率8%:
  NPV = Σ(第n年现值) - 初始投资

投资回收期:
  PB = 初始投资额 / 年均净收益 (年)
```

### B. 术语表

|-----|------|
| **储能** | 将电能存储后再释放的技术 |
| **峰谷电价** | 高峰和低谷时段的电力价差 |
| **调度** | 电网对能源的分配和管理 |
| **IRR** | 内部收益率,投资的真实回报率 |
| **NPV** | 净现值,考虑时间价值的收益评估 |
| **容量** | 装置的额定功率输出 |
| **能量** | 实际可存储的电能量 |

### C. 常见问题

**Q: 计算结果准确吗?**
A: 本工具基于江苏省当前政策和市场数据,结果用于参考评估。实际项目需考虑地方政策变化。

**Q: 支持导出我的计算结果吗?**
A: 支持导出PDF报告、Excel表格和JSON数据。

**Q: 我的数据会被保存吗?**
A: 仅在浏览器本地保存 (localStorage)。刷新页面或关闭浏览器后数据清空。
