## 🎯 网站整体设计梳理

### 核心功能设计

#### 1. 主功能 - 储能项目收益计算

本工具是一个**专业的财务计算器**，用于快速评估独立储能项目的经济收益：

```
用户输入 → 参数验证 → 收益计算 → 结果展示 → 报告导出
```

**计算场景**:
- ☀️ **夏季峰谷套利** (6月-9月高温期)
- 🌙 **非夏季调度** (其他月份基础调度)
- 💰 **政策补贴** (政府激励政策)
- 📊 **综合评估** (多维度投资分析)

#### 2. 用户交互设计

| 功能模块 | 设计特点 |
|---------|---------|
| **参数输入** | 表单型设计,支持实时编辑和验证 |
| **实时计算** | 无延迟响应,毫秒级计算完成 |
| **结果展示** | 多维度图表和数据可视化 |
| **报告导出** | PDF/Excel/JSON多格式支持 |
| **多语言** | 40+种语言完全本地化 |
| **响应式** | 桌面/平板/手机完美适配 |

### 数据流设计

#### 用户数据流

```
1. 用户访问入口页面
   → 浏览器自动检测语言偏好
   → 跳转到相应语言版本

2. 用户输入项目参数
   → 前端实时验证
   → 本地缓存输入值 (localStorage)

3. 触发计算
   → 调用业务逻辑计算模块
   → 完全在客户端执行 (无服务器)

4. 结果展示
   → 渲染可视化图表
   → 显示详细数据分析

5. 用户操作
   → 导出报告 (PDF/Excel)
   → 分享结果 (URL编码)
   → 调整参数 (重新计算)
```

#### 计算数据流

```
原始参数 → 标准化处理 → 业务验证 → 核心计算 → 敏感性分析 → 结果聚合 → 输出展示
```

### 核心计算模型

#### 关键公式

**年度总收益 = 夏季收益 + 非夏季收益 + 政策补贴**

```
夏季收益 (¥)
= 装机容量(MW) × 
  峰谷价差(¥/MWh) ×
  储能时长(MWh) ×
  充放电效率(%) ×
  夏季运营周期数

非夏季收益 (¥)
= 装机容量(MW) ×
  基础价格差(¥/MWh) ×
  储能时长(MWh) ×
  充放电效率(%) ×
  非夏季运营周期数

年度净收益 (¥)
= 总收益 - 运维成本 - 税费
```

#### 投资评估指标

| 指标 | 说明 | 应用 |
|-----|------|------|
| **IRR** | 内部收益率 | 评估投资真实回报 |
| **NPV** | 净现值 (8%折现率) | 绝对经济效益评估 |
| **静态回收期** | 投资回本年限 | 简易风险评估 |
| **收益率** | 年收益/投资额 | 快速对标参考 |
