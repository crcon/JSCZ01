# 贡献指南

感谢您对「江苏独立储能收益测算模型」项目的关注和支持!

## 如何参与

### 提出Issue

发现bug或有改进建议？请在 [GitHub Issues](../../issues) 中告诉我们。

**提交Issue前请:**
- 检查是否已有相同的Issue
- 提供清晰的问题描述和重现步骤
- 附上错误截图或日志信息

### 提交Pull Request

我们欢迎您提交Pull Request来改进这个项目。

**提交PR前请:**
1. Fork本项目
2. 创建功能分支 (`git checkout -b feature/amazing-feature`)
3. 提交您的更改 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建Pull Request

### 代码规范

- 遵循现有代码风格
- 添加必要的注释和文档
- 确保代码通过现有的linter检查
- 为新功能添加测试用例

### 文档贡献

帮助改进文档也是很重要的贡献!

- 修正拼写错误
- 改进文档清晰度
- 添加使用示例
- 翻译文档

## 开发流程

### 本地设置

```bash
# Fork后克隆您的仓库
git clone https://github.com/your-username/JSCZ01.git
cd JSCZ01

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

### 测试

```bash
# 运行测试
npm run test

# 生成覆盖率报告
npm run test:coverage
```

### 构建

```bash
# 构建生产版本
npm run build

# 导出静态文件
npm run export
```
